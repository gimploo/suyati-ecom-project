# Ecommerce plugin  (Suyati Project)

### Steps to run the website

### 1. Run the backend
1. cd ecom_backend/
2. pip install -r requirements.txt
3. python manage.py runserver

### 2. Run the frontend
1. cd front-end/
2. npm install 
3. npm start 

### 3. ML 
1. cd ML/
2. python ML_CollabFiltering.py
